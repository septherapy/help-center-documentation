---
title: Meeting controls
---
Along the bottom of the therapy room are the meeting controls. The placement is the same for all participants in the therapy room. While the basic functionality is the same as well, only providers have the ability to end a session for all participants while students can only leave the session.

<Card title="Microphone controls" horizontal="false" target="_blank">
  <Image src="https://blob-cdn.documentation.ai/org-2298b9ec-a237-4db7-8e7e-dee229ecc015/doc-f231f1a4-ce9f-444f-aa94-d0fc4be71883/1773173507406-qbzq45pqjae-Microphone-button.Microphone button?q=85&fm=auto&auto=compress%2Cformat" width="46" height="46" alt="" style="float: right; margin: 0 0 15px 0;" />

  Allows you to unmute or mute your microphone if one has been detected.
</Card>

<Card title="Primary camera controls" horizontal="false" target="_blank">
  <Image src="https://blob-cdn.documentation.ai/org-2298b9ec-a237-4db7-8e7e-dee229ecc015/doc-f231f1a4-ce9f-444f-aa94-d0fc4be71883/1773173507406-qbzq45pqjae-Microphone-button.Microphone button?q=85&fm=auto&auto=compress%2Cformat" width="46" height="46" alt="" style="float: right; margin: 0 0 15px 0;" />

  Allows you to turn on or off your primary camera if one has been detected.
</Card>

<Card title="Document camera controls" horizontal="false" target="_blank">
  <Image src="https://blob-cdn.documentation.ai/org-2298b9ec-a237-4db7-8e7e-dee229ecc015/doc-f231f1a4-ce9f-444f-aa94-d0fc4be71883/1773173507406-qbzq45pqjae-Microphone-button.Microphone button?q=85&fm=auto&auto=compress%2Cformat" width="46" height="46" alt="" style="float: right; margin: 0 0 15px 0;" />

  Allows you to turn on or off your document camera if you have one connected. You can select a document camera in the device settings dialog.
</Card>

<Card title="Device settings" horizontal="false" target="_blank">
  <Image src="https://blob-cdn.documentation.ai/org-2298b9ec-a237-4db7-8e7e-dee229ecc015/doc-f231f1a4-ce9f-444f-aa94-d0fc4be71883/1773173507406-qbzq45pqjae-Microphone-button.Microphone button?q=85&fm=auto&auto=compress%2Cformat" width="46" height="46" alt="" style="float: right; margin: 0 0 15px 0;" />

Shows the device settings dialog box which allows you to choose what devices are active or used for the current session. From here you can change the following:

- Primary camera

- Document camera

- Microphone

- Speakers
</Card>

<Callout kind="info" collapsed="false">
  If you are having trouble with changing or recognizing your devices, please see the [Troubleshooting](/providers/troubleshooting) guide.
</Callout>

<Card title="End session" horizontal="false" target="_blank">
  <Image src="https://blob-cdn.documentation.ai/org-2298b9ec-a237-4db7-8e7e-dee229ecc015/doc-f231f1a4-ce9f-444f-aa94-d0fc4be71883/1773173507406-qbzq45pqjae-Microphone-button.Microphone button?q=85&fm=auto&auto=compress%2Cformat" width="46" height="46" alt="" style="float: right; margin: 0 0 15px 0;" />

  Allows you to end the session. Upon clicking the button, you will have the following options:

- Stay in therapy room - All participants will be dismissed from the session while you remain in the therapy room. Primarily used to end the current session and admit a participant to start a new session.

- End session and exit - You and all participants will be dismissed from the session. Providers will be redirected to their dashboard while students will be redirected to a post-session page.
</Card>
